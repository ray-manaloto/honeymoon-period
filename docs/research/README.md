## Evidence labels

- **Verified** — directly supported by current source, primary documentation, an explicit user decision, or a retained test artifact.
- **User-observed** — reported during testing but not yet backed by a public fixture, screenshot, or log.
- **Provisional** — inherited from an earlier plan and requires fresh primary-source verification.
- **Needs refresh** — temporally unstable or missing retained evidence.
